# Rubygems Tracker

Track gems download statistics.

# Basic info

[https://blog.openshift.com/yeoman-serves-up-a-real-mean-stack/](https://blog.openshift.com/yeoman-serves-up-a-real-mean-stack/)

# Usage

## Dependencies

    $ brew install mongodb

    # run mongodb
    $ mongod

## Development

    $ npm install
    $ bower install

    # run server
    $ grunt serve
